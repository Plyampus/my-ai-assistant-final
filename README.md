## English Version

### Project Description
This project is an AI assistant designed to interact with GitHub data, providing intelligent responses and executing various tasks related to repository management, issue tracking, pull requests, and more.

### Features
- Intelligent interaction with GitHub repositories
- Ability to create, update, and manage files and branches
- Support for pull requests and issue tracking
- Enhanced user experience through automation and intelligent suggestions
- Mobile responsive design
- Real-time chat interface

### Architecture
The architecture consists of a front-end interface (Next.js + React) interacting with the backend (Node.js + Express) which connects to Google Gemini API for AI capabilities. The system stores chat history in JSON format locally.

### Quick Start
1. Clone the repository: `git clone https://github.com/Plyampus/my-ai-assistant-final.git`
2. Navigate to the backend: `cd backend && npm install && npm start`
3. In another terminal, navigate to frontend: `cd frontend && npm install && npm run dev`
4. Open browser at http://localhost:3000

### Technologies
- Frontend: Next.js, React, Tailwind CSS, Axios
- Backend: Node.js, Express.js, Google Gemini API
- Storage: JSON files for chat history
- Deployment: Vercel (frontend), Render (backend)
